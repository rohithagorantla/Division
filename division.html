HTML

                HTML is not a programming languague, it is a markup languague.
                HTML stands for Hyper Text Markup Languague.
                HTML was developed by Tim-Berners-Lee.
                It is derived from SGML(Standard Graphics Markup Languague).
                HTML is case insensitive languague.
                HTML is very simple and logical.
		It is stated to be known as backbone of webpages.

		

                HTML is divided into tags, attributes and elements.
                HTML document contains HTML tags and plain text.
                It uses markup tags to describe webpages.
                Each page has series of links to other pages called Hyperlinks.
                HTML file has an extension .htm or .html.
                Currently we are using HTML5 version.
                HTML is used for web applications.
                It is platform independent languague.
